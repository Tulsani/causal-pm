# Causal Engine

Builds product and causal graphs from DOM structure, event streams, outcomes, and assumptions.

Initial responsibilities:

- reconstruct session journeys
- build product structure graphs
- build behavioral sequence graphs
- combine both into candidate causal graphs
- estimate simple treatment effects
- answer counterfactual queries with explicit caveats

The early version should prefer transparent assumptions over complex causal discovery.

See `USAGE.md` for the first runnable Synerise dataset prototype.
